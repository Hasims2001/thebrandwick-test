## Installation
To set up the project locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/Hasims2001/thebrandwick-test.git
```

2. Open the coding-test folder in termial:

3. Run command:
```
npm install
```

4. Run project
```
npm start
```
